Voici les étapes pour générer l'application modèle de Flutter :

Ouvre ton terminal dans le dossier où tu souhaites stocker tes projets 📁.

Exécute la commande suivante pour créer l'application :

flutter create mon_app
Remplace "mon_app" par le nom que tu souhaites donner à ton application.

Comment lancer l'application sur un appareil ou un émulateur ?
Pour voir les appareils connectés : flutter devices
Pour lancer l'application sur un appareil spécifique : flutter run -d nom_de_l_appareil
Pour voir les émulateurs disponibles : flutter emulators
Pour lancer un émulateur : flutter emulators --launch nom_de_l_emulateur
